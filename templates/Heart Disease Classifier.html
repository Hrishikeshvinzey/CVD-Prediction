How this Heart disease Predection works?

Heart disease Dataset

Medical report data is collected from various rescources,to form a dataset which includes data of both people suffering from heart disease and those who are healthy.Based on previously collected real dataset Machine Learning model is trained.

Machine Learning Model

Using basic models like Decision Tree, Support Vector and Random Forest Algorithm we have elevated the workings and developed three more Machine Learning models, Ensemble Modelling, Random Forest Hyper tuning and a tuned XGBoost model which is trained on collected medical dataset.

Make Predictions

On entering any individual's medical data as input to ML model it predicts the individual as suffering from heart disease or is healthy.
